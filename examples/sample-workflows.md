# Sample workflows

## Public reusable skill workflow

1. User makes a broad request.
2. Agent identifies likely paths.
3. Router selects the narrowest correct path.
4. Agent executes or delegates.
5. If blocked, fallback stays close to original intent.

## Install/adopt workflow

1. Clone repo.
2. Read `README.md`.
3. Copy or link `skill/` into your skill directory.
4. Review `SKILL.md` triggers.
5. Adjust references/examples for your environment.
